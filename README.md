# 🌿 Leif — Personal AI Assistant

Leif is a custom-built, highly personalized AI assistant designed from scratch. She acts as a pair programmer, business strategist, and fitness coach. Currently in Phase 1, Leif operates via a custom Command Line Interface (CLI) powered by the Anthropic Claude API.

## 🚀 Features (Phase 1)
- **Custom Personality:** Witty, direct, and genuinely supportive. Leif is designed to feel like a sharp co-founder rather than a robotic assistant.
- **Teaching-First Code Assistance:** Leif doesn't just write code; she explains concepts line-by-line to build real development skills.
- **Stateful Memory:** Maintains conversational context throughout the active session.
- **Graceful Error Handling:** Custom error catching for rate limits, API timeouts, and missing credentials.

## 🛠️ Tech Stack
- **Language:** Python 3.12
- **AI Engine:** Anthropic Claude 
- **Environment:** Secure virtual environment (`venv`) with `python-dotenv` for local secret management
